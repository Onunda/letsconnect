## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
### To fix vulnerabilities and check updates
```
npm i -g npm-check-updates
```

### Install DevTools to your browser
See [chrome](https://chrome.google.com/webstore/detail/vuejs-devtools/nhdogjmejiglipccpnnnanhbledajbpd)

See [firefox](https://addons.mozilla.org/en-US/firefox/addon/vue-js-devtools/)

####As an option can add the view standalone app
```
npm install -g @vue/devtools
```
and call it by running
```
vue-devtools
```
you can the add the following script tag in index js to view app in standalone app
```
<script src="http://localhost:8098"></script>
```
